# O-DI-GA

### 프로젝트 소개
- 실시간 웨이팅 및 예약부터 결제까지 사용자 앱 하나로 이용 할 수 있어 매장 사용자에게 편리함을 제공하고, 매장 관리자는 가게 관리 및 매출 테이터 기반 통계 및 분석으로 가게 운영을 한번에 할 수 있는 통합 플랫폼 **O-DI-GA**
